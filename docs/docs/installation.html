# Installation

If all prerequisites are met, installing Chariot.js is as easy as initializing a new npm project by typing either npm init and running npm install chariot.js. If you are using a version of npm lower than 4.5 (you shouldn't) run following command instead which saves Chariot.js as a dependency in your package.json file: npm install chariot.js --save. This will install the latest stable version of Chariot.js into your freshly created project.
